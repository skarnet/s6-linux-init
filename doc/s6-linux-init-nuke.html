The s6-linux-init-nuke program

s6-linux-init-nuke kills all processes immediately.

Interface 

     s6-linux-init-nuke

s6-linux-init-nuke sends a SIGKILL to every process on the system. (Process 1 will be unaffected.)

Do not try this at home.

Notes

  • This program only exists so the s6-linux-init package can be self-contained and avoid relying on the existence of an external kill program. It is only run in the very improbable case that s6-svscan crashes, in a container situation.
